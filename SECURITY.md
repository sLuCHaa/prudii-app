# Security Policy

Prudii is a privacy-first email client — security reports are taken seriously.

## Reporting a vulnerability

Please **don't** open a public issue for security problems. Email me directly:

**slucha88@gmail.com**

Include what you found, how to reproduce it, and the impact if you know it. I'll get back to you as soon as I can and let you know once it's fixed. Please give me a reasonable window to ship a fix before disclosing publicly.

## Scope

This repository is the desktop client (Tauri + React). In scope: local data handling, credential storage, IMAP/SMTP/OAuth handling, license verification, tracker blocking, and anything else in this codebase.

## Supported versions

Only the latest release receives security fixes.
